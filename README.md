perl練習用のリポジトリです

以下は作業時に使用する。 
ドキュメントの参照  
* perldoc perlre	：正規表現のドキュメント
* perldoc perlsyn	：文法のドキュメント
* perldoc perlop	：演算子のドキュメント
* perldoc perlfunc：組込み関数のドキュメント
* perldoc utf8	：utf8プラグマのドキュメント
* perldoc -f 関数名：その関数のリファレンスをすぐに開く
